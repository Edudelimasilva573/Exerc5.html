Descrição do curso

Este curso apresenta conceitos básicos de web e mostra as partes e funções das tags em um documento HTML. Ensina, também, a formatar uma página. Inserindo imagens e muito mais!tudo isso, com explicações simples e de fácil compreenção.

Estrutura do curso

  1. Módulo 1 - Introdução
  2. Módulo 2 - html e a web
  3. Módulo 3 - Criando um documento HTML
  4. Módulo 4 - Formatando um documento HTML
  5. Módulo 5 - Criando listas
  6. Módulo 6 - Inserindo imagens
  7. Módulo 7 - Criando links
  8. Módulo 8 - Combinando tags
  9. Módulo 9 - Usando imagens mapeadas
